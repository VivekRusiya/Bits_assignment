# Distributed and Parallel Training of CBOW using Parameter Server

## Overview
This project implements distributed training of the Continuous Bag of Words (CBOW) model using a parameter-server-based asynchronous architecture.

## Features
- Data-parallel CBOW training
- Asynchronous parameter server
- Multiprocessing-based communication
- Speedup and throughput analysis
- Communication cost evaluation

## Dataset
Project Gutenberg ebook corpus:
